## 🕰️ 진행 방식

### 📌 매일 아침
- **시간:** 오전 08:00
- **과제:** 전날 배운 내용을 활용하여 **백준 문제 풀이**
- **지각 기준:** 08:15:00 이후

### 📌 저녁 코드리뷰
- **시간:** 18:30까지
- **리뷰 방식:**
  - 서로 코드 리뷰 남기기
  - ChatGPT에게 `"내장 함수나 메서드를 이용한 다른 방법으로 풀어줘"` 요청하여 **다양한 풀이 습득**
  - 리뷰는 `reviews/멤버명/문제번호_review.md` 에 기록

## 🧠 커밋 컨벤션

| 태그 | 설명 |
|------|------|
| `feat` | 새로운 문제 풀이 추가 |
| `review` | 코드 리뷰 |
| `docs` | 문서 수정 |
| `refactor` | 코드 리팩터링 |


> 예시:
> ```
> feat: [week1] 정희 - 1000번 문제 풀이
> review: [week1] 새하 - 정희 1000번 리뷰 완료
> ```
